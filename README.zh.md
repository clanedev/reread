# reread

[English](/README.md) | 中文

reread 是一个本地优先的 EPUB 阅读器。它更像一个轻量的本地阅读工具：你把书放在自己的文件夹里，打开网页就能读，不需要云端账号，也不需要把书重新上传到别的服务。

## 你什么时候会用到它

如果你有下面这些需求，reread 可能会比较适合你：

- 你已经把电子书保存在本地，想直接在网页里阅读
- 你希望浏览器能直接访问你选择的本地文件夹
- 你不想依赖在线书城或同步服务
- 你希望刷新页面后还能回到上次读到的位置
- 你想把书库、封面、进度和阅读设置都放在一个简单的本地应用里

## 它现在能做什么

- 直接打开本地 EPUB 文件
- 用 File System Access API 访问用户选择的本地文件夹
- 记住你上次打开过的书
- 保存阅读进度和阅读设置
- 通过 shelf 管理本地书库
- 用适合 EPUB 的方式恢复阅读位置

## 当前功能

- 本地文件夹选择和书库浏览
- EPUB 封面预览和书籍信息缓存
- 阅读器页面和目录导航
- 阅读进度保存与恢复
- 字体大小、行距、段落间距、主题等阅读设置
- 基于本地存储的书籍索引

## 技术概览

- Next.js App Router
- File System Access API
- IndexedDB 用于保存书籍元数据和阅读进度
- foliate-js 用于 EPUB 解析和渲染
- zip.js 用于读取 EPUB 这种 ZIP 结构的文件
